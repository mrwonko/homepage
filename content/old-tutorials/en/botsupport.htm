>> Bot Support - Easy

The easiest way to make bot-support for multiplayer-maps
-----------------------------------------------------------------------------------------------------

All in one, it's pretty easy to make botsupport:

1. First, you need a shortcut or an BAT-File to start your JK2-MP im Wypoints-Edit-Mode:

">hier dein jk2-Ordner<\GameData\jk2mp.exe" +devmap mapname +set bot_wp_edit 1

"mapname" is the name of your map.

2. Now you can start JK2-MP with you Shortcut / Bat-File.

3. After your map is loaded, drop down the Console and type:

/bot_wp_save

4. As next you must bind any key with command to make "waypoints, like in this example:

/bind i "bot_wp_add"

with this statement you have bound the command "bot_wp_add" to "i"-key. It means, if you press "i", a new waypoint will be set at position of player.

5. Now you can run through you map und press the "i" key to set the waypoints. You can see all waypoints as colored Digits.

You must make sort of Botway with this funktion. It is better to make closed route, it means, that the last waypoint ist close to the first ...

If you have to delete any waypoint just type:


"/Bot_wp_rem"

or

"/Bot_wp_rem" "point-nr."


6. If you did set all waypoints and your botpath is complete, just type:

/bot_wp_save

with this statement your "botroute" will be save to file.7. Arfter this, you can end JKII and begin to search for your route-file. It will be stored in "base/botroutes" normally and names "mapname.wnt", where "mapname" means name of your map.

so, it's all you must done...

Note There are many more commands and options for making bot-support, but the are not really needed, and you should use them only, if you are sophisticated in making multiplayer maps and Botroutes.
