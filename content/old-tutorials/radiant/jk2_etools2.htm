>> Jedi Knight II - Editing Tools II <<

 

SCHRITT 1: Die Lizenzvereinbarung.

Um die "Editing Tools 2" zu installieren, muss man die Lizenzvereinbarung akzeptieren. Um das zu tun, einfach auf [ OK ] klicken.

Wenn du mit der Lizenzvereinbarung nicht einverstanden bist, klicke auf [ Abbrechen ]. In diesem Fall werden die Tools nicht installiert und du kannst hier aufhören.

SCHRITT 2: Installationspfad

Hier wird ein Standardordner vorgeschlagen. Es ist sehr wichtig, dass du diesen Pfad mit deinem richtigen Jedi-Outcast/GameData Installationspfad ersetztst.

Klicke auf  [ Browse... ] und finde deinen Installationspfad. 

Wichtig: wähle unbedingt den "GameData"-Ordner !

SCHRITT 3: Die Installation

Nach dem du den richtigen Pfad für die Installation eingestellt hast klicke auf [ Unzip ] um die "Editing Tools 2" zu installieren.

 

SCHRITT 4: Installation Abschluss

Nach dem alle Dateien installiert wurden, erscheint diese Dialogbox. 

Klicke auf [ OK ] um die Installation abzuschließen
